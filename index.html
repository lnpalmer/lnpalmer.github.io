About me

For the past few years I've been learning about computer science and software development. I dabbled in a few fields and then studied game development for a couple years; I only released one complete game but along the way learned a lot about graphics, parallel processing, GPUs, and general design principles. I recently took an interest in deep learning and have been shifting towards hands-on work with neural nets. I interned at Hyland Software over the summer of 2016; I worked in the development department and completed several software projects. I'm majoring in computer science at Ohio University (class of 2020).

During my game development work, I used OpenGL and other 3D graphics APIs along with several game engines. Languages I use for game development include Lua and C# inside engine environments, and C++ and JavaScript (CoffeeScript) on their own. Using the Defold engine I made Labyrinth, a minimal, retro dungeon crawler with a procedural map and a lot of focus on UI design.

I've been studying deep learning for the past few months. To strengthen my understanding of neural networks both in theory and in practice, I'm working on rnet, a simple program that uses a recurrent neural network to build a character-level language model based on an input text file. The entire program is written from scratch in C++; GPU acceleration is done with OpenCL.
